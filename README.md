## Installation Steps 🛠️

---

<br>

### Install Dependencies ⚙️

```bash
yarn install
```

### Run Test Suite 🧪

```bash
yarn test
```

### Launch Application In Development Mode ⌨️

```bash
yarn dev
```

### Build Production Version 🏗️

```bash
yarn build
```

### Launch Application in Production Mode _(Run after Build Step)_ 🚀

```bash
yarn start
```
